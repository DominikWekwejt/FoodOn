## General info
This project is simple app for ordering or delivery food.

## Technologies
Project is created with:
* Android Studio v2023.1.1
	
## Setup
To run this project, install it locally using npm:
<br>
back:
```
$ npm install
$ npm start
```
front:
open new terminal(Ctrl+~)
```
$ cd FoodOn
$ npm start
```
